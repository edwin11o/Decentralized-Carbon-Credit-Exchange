# Decentralized Carbon Credit Exchange

A blockchain-based marketplace for trading verified carbon credits on the Stacks blockchain.

## Features
- Issue verified carbon credits
- Create marketplace listings
- Purchase and trade credits
- Retire credits for environmental impact
- Reputation system for users
- Platform fee management

## Smart Contract Functions
- `issue-carbon-credits`: Issue new verified credits
- `create-listing`: List credits for sale
- `purchase-credits`: Buy credits from listings
- `retire-credits`: Permanently retire credits
- `cancel-listing`: Cancel active listings

## Setup
1. Install Clarinet: `brew install clarinet`
2. Clone this repository
3. Run `clarinet check` to verify contract
4. Run `clarinet test` to execute tests
